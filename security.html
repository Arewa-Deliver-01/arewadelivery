Security Overview

Overview of the technical and operational security measures we use to protect users and data.

Technical Security

Encryption (HTTPS/TLS), secure storage, and least-privilege access policies protect user data.

Operational Security

Background checks for riders where applicable, secure onboarding for vendors, and internal access controls for admin staff.

Reporting & Response

Security incidents are logged and investigated. If you suspect a security issue, contact security@arewadeliver.example.
